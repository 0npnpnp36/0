Privacy

Last updated: 14 July 2026

This page describes how the affection site (“the site”) handles information. The site is a front-end experience (sound and graphics). It does not require an account.

What we collect

Right now the site does not run its own backend and does not ask you for a name, email, or login.

  • Interaction stays in your browser (for example pointer position for visual effects, and whether sound is on).
  • Audio and fonts may be loaded from this site or from third-party CDNs (for example Fontshare / Google Fonts). Those providers may see normal technical request data such as IP address, as with any website.
  • If the site is hosted on GitHub Pages or a similar host, that host’s logs may include standard server request data.

Cookies & storage

The site does not use advertising cookies. Any preference kept in the browser (if added later) would be for the experience only, not for cross-site tracking.

If a backend is added

If affection later stores data on a server (for example submissions or analytics), this page will be updated to say what is stored, why, and how long it is kept.

Contact

For privacy questions about this site, contact the project maintainer through the repository or site owner.
